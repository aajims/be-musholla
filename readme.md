## Documentation

1. git clone https://github.com/aajims/be-musholla.git
2. cd be-musholla
3. composer install
4. php -r "file_exists('.env') || copy('.env.example', '.env');"
5. php artisan key:generate
6. php artisan migrate -s
7. php -S localhost:8000 -t public
